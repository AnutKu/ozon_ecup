## Задача

Разработать ML-решение, которое по описанию и метаданным товара определяет, является ли он контрафактом. Цель — защита клиентов и репутации Ozon от поддельных товаров на платформе.

## Структура репозитория

Файлы **не связаны между собой** и представляют собой независимые подходы к решению задачи, которые мы использовали на хакатоне.

| Файл | Описание |
|------|----------|
| `ozon_preprocessing.ipynb` | Подход с упором на предобработку данных и feature engineering |
| `ozon_model.ipynb` | Подход с упором на подбор и обучение модели |
